RESIN CRAFT SHOP WEBSITE
    This is a website made for our own resin craft small business.
    You can look at products, check for prices and even make custom orders. its made to be responsive on all devices(mobile phones, tablets and desktops).

FEATURES
• Sticky header with navigation bar and logo
• Hero section with page name and subtitle
• Grid layout for products with hover effect
• Available color options for add-ins
• Custom Order contact form 
• FAQ section
• Back to top button
• Fully responsive design for all screen sizes
• Footer with business info

TECHNOLOGIES USED
• HTML
• CSS(with custom fonts and colors)
• JavaScript (for dropdowns and interactive buttons)
• Pexels and Pinterest (Images)
• Iconscout (Icons)
• FormSubmit (Contact Form Handling)
• Sunroll Font (Logo)


How to Run It
1. Download or clone the project
2. Open index.html in your browser
3. Or check the live version here: https://harlyresins.netlify.app/

How to Use
• Click on the hamburger menu or buttons on the navigation bar to go to different pages
• Hover on product cards to see effects
• Use the color and add-ins list and dropdown to see available color and add-in options
• Use the custom order to send inquiries, place custom orders and give comments.
• Check out the FAQ section in the contact page to get answers on frequently asked questions.
• Use the Back to Top button in the catalog page to move to the top faster
• Check the contact page or the contact section in the homepage to reach us out on our socials or email us.
